About the Forest Fire Prediction Model

This application predicts the risk or intensity of forest fires using a trained Machine Learning model. It considers multiple weather and environmental factors that significantly influence the occurrence of fires.

🔑 Input Features:

  • 🌡 Temperature
  • 💧 Humidity (RH)
  • 🌬 Wind Speed
  • 🌧 Rain
  • 🔥 FFMC
  • 📊 DMC
  • ⚡ ISI
  • 🏷 Classes
  • 🌍 Region

All inputs are preprocessed using a Standard Scaler, then passed into a Ridge Regression model which provides accurate predictions for fire risk. This system supports decision-making for early fire prevention and forest management.
